# Phase 0: Prerequisites

## Linear Algebra (Most Important)

Quantum computing is fundamentally linear algebra.

### Learn:

- Vectors
- Matrix multiplication
- Eigenvalues and eigenvectors
- Inner products
- Orthogonality
- Unitary matrices
- Tensor products

### Key Insight

A qubit is a vector:

|ψ⟩ = [ α ]
[ β ]

### Checklist

- [ ] Understand vectors
- [ ] Understand matrix multiplication
- [ ] Understand eigenvalues and eigenvectors
- [ ] Understand orthogonal vectors
- [ ] Understand complex vector spaces
